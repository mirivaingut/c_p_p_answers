# C++ Course Answers Repository

This repository contains answers to exercises and course questions for C++.

## What is inside

- dated folders with separate exercise sets
- Visual Studio projects and C++ source files
- example solutions for course tasks

## Folder overview

- `05.04.22/` - contains a `Radar` Visual Studio solution with files for a radar exercise.
- `24.03.22/` - contains several standalone C++ files with short examples and problem answers.
- `29.03.22/` - contains threading examples in a `th1` Visual Studio solution.
- `30.03.22/` - contains multiple exercise folders for specific course questions, each with their own solution project.

## Note

This repository is a collection of answers and examples, not a single unified application.
